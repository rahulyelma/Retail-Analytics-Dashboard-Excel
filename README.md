# Retail Analytics Dashboard (Excel)

## Project Overview

This project demonstrates the creation of an interactive Retail Analytics Dashboard using Microsoft Excel.

The dashboard provides insights into sales performance, profitability, customer segments, and regional performance through KPIs, Pivot Tables, Pivot Charts, and Slicers.

## Objectives

* Analyze retail sales performance
* Track profitability across regions
* Compare sales across product categories
* Provide interactive filtering using slicers

## KPIs

* Total Sales
* Total Profit
* Total Orders
* Profit Percentage

## Dashboard Components

### Sales by Category

Displays revenue generated by different product categories.

### Profit by Region

Shows profit contribution from different geographical regions.

### Sales by Segment

Analyzes sales distribution among customer segments.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Dashboard Design

## Key Insights

* Technology category generated the highest sales.
* Regional performance can be compared using profit analysis.
* Customer segments contribute differently to overall revenue.
* Interactive slicers allow dynamic exploration of the data.
